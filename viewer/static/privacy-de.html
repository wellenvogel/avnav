AvNav Datenschutzerklärung

 

AvNav kontaktiert während des Betriebes keine Server im Internet ausser für die demo Karten.

 

Weitere Ausnahme: Wenn in den Einstellungen als NMEA/AIS Quelle “IP” gewählt wird, kontaktiert AvNav den dort angegeben Server (IP und Port) um Standort Daten und AIS Informationen zu erhalten.

Normalerweise wird sich ein solcher Server im lokalen Netz (WLAN) befinden. Die Ip Adresse muss vom Nutzer explizit eingegeben werden.

 

Während des Betriebes erfasst AvNav die Standort-Daten und schreibt sie (als Track) Daten in das gewählte Verzeichnis auf dem Gerät oder der SD Karte.

Diese Daten bleiben dort so lange erhalten, bis der Nutzer sie löscht.

 

Sonstige personenbezogene Daten werden durch AvNav nicht erfasst oder benutzt.

 

AvNav benötigt die folgenden Zugriffsrechte:

Netzwerk

Für den Zugriff auf einen Server für Standort/AIS Daten (Serveradresse wird durch den Nutzer festgelegt).

Im Mode “external Browser” kann erlaubt werden, dass sich weitere Geräte mit avnav verbinden (in den Einstellungen).

Bluetooth

Für den Zugriff auf GPS/NMEA Sender.

Schreibrechte für externen Speicher

Der externe Speicher kann als Arbeitsverzeichnis gewählt werden.

Standort Informationen

Das wird für die Grundfunktionen der Positionsdarstellung benötigt.
