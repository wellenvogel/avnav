The Settings Page

From the  main page you get here using the button.

Buttons

Icon Name Function

SettingsOK accept changes and go back

SettingsDefaults reset all settings to their default values

SettingsLayout start of the layout editor

SettingsAddons go to user app configuration

MOB man over board (see main page)

Cancel discard changes and go back

You can set a lot of parameters that influenc eyour display. All those setting swill be stored inside the current browser for the current URL you sue to reach AvNav. They are not stored on the server!

For most of the values there is a separate reset to change to the default. Globally you can reset to defaults using the button. If you are going to leave this page without saving you will be prompted.
