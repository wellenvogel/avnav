The AIS List

From the Ais Info you get here using the button.

Buttons

Icon Name Function

AisNearest back to "normal mode", on the  navigation page the nearest target will be shown
AisSort change the sort order of the list

MOB man over board (see main page)

Cancel back to last page

On this page you will see all targets inside a circle of 10nm to the current position or to the center of the chart(sorted by CPA). By clicking a row you will go back with the chart centered to this target.

By clicking the second headline row or the button you can change the sort order.
