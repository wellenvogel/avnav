The Route Editor

Overview

You will reach the route editor with the  button from the navigation page . Additionally you can reach it via the Edit button within the info dialog for a route on the Files/Download page.

At this page you can create and edit a route.

At each point in time there is a currently selected point within the route. It is marked red on the chart and marked grey in the list on the left side. You can change the selected point by clicking on a different point on the chart or in the list on the left side.
If you are editing the currently active route, the route info at the left side will have a red border (otherwise green).
If you leave the editor the display will go back to the active route.

All changes will become effective immediately - there is no fallback.

At the left side you have the list of all points in the route with courses and distances. In the upper display you have the overall route length and the estimated ETA (if the route is active).
By clicking on an already selected waypoint in the list you can open an edit dialog to change the name or position.

Route Dialog

Clicking on the route info on the left side willbring up a dialog.

Within this dialog you can change the name of the route, you can select a new route to be edited, you can copy the route, delete it or empty or invert it.

Clicking Edit will move you to the route list. At this page you can delete and edit all points, change the name, invert or load a new route.

Clicking on the lower left panel will bring up some additional buttons.

Using those buttons you can move between the points in the route or you can edit the selected point.
Clicking on the lower right will center the chart to the boat position.

Using Overlays

If there is an overlay visible (like in the picture) - or within ocharts you can bring up a new dialog by clicking on an object on the chart. This dialog will provide some additional functions.

You can now insert the point you have clicked into the route (befor or after the current red point). This way you can easily e.g. use points from a waypoint file to create your routes.

If the overlay is a route you can insert the part after the clicked point into your current route.

The dialog will show you the selected start point of the route and you can decide to insert before or after your current red point.
