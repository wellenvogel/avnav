Route List Page

Overview

You can reach this page by clicking "Edit" in the Route Dialog shown at the Route Editor.

Buttons

Icon Name Function

RoutePageOk Save changes and back to Route Editor.

NavGoto Start navigation to the currently selected waypoint (the one with a grey background).

NavInvert Invert the route.

NavDeleteAll Empty the route

RoutePageDownload Open the  Files/Download Page section routes.
You can select a route to be edited - or upload/download routes. After selecting a route there you will return to this page.

MOB Man over Board(see Mainpage)

Cancel Cancel (discard changes) and go back to the Route Editor

You can edit all the points in the route or create a copy of the route. By clicking on the route name you will get a dialog.

  After "Ok" you can return to the Routen Editor to edit your new route.
