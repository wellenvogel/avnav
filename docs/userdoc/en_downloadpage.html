The Files/Download Page

From the mainpage you will get here using the button. At this page you can download/upload/delete/edit/view charts,tracks,routes, user files and images.

Buttons

Icon Name Function

DownloadPageCharts show list of charts

DownloadPageTracks show list of tracks

DownloadPageRoutes show list of routes

DownloadPageLayouts show list of layouts

DownloadPageUser show list of user files

DownloadPageImages show list of user images
DownloadPageOverlays show list of overlay files

DownloadPageUpload upload a file to the currently show category

Cancel back to previous page

For each entry in the list there will be a couple of informations and there is a delete and a download button. They are only visible if the operation is permitted for the particular element (e.g. no deletion of demo charts or system layouts).
Potentially there will be more icons to give you a hint which operations are possible for this element. By clicking on the element a dialog will be shown with the available actions for the element.

If available ("Edit" / "View") you can move to an edit/show view. For routes "Edit" will bring up the Route-Editor.

In this view you can make changes - if "Editing" is shown or the content is shown. By clicking on the changes will be stored.

Special Functions

User Files

By changing to user files you can e.g. edit user.js and user.css. The file keys.json contains user defined keyboard shortcuts.

The file images.json can be used to adapt some of the symbols in AvNav.

Ypu can also upload additional files here that you need to adapt AvNav (e.g. css files, java script files or HTML files). Those files you can reach via the url /user/viewer/<name>.
Images should be uploaded using the Images category. They will be reachable via /user/images/<name>.

Using the "+" button (lower right) you can create a new file. This could be an HTML file that you later on would use to create a User App .

For a HTML file the action dialog will contain an "App" button that you can use to directly create a User App. You have to be sure to upload an Icon image file beforehand.

Charts - mbtiles Format

Using a chart in mbtiles could lead to problems if the tiles internally are not ordered using the default "xyz" format and there is no hint in the meta data about that. If you experience display problems with an mbtiles chart you can try to switch the format in the action dialog to "tms". This change will be written to the chart file.

Chart Upload

At the category you can directly upload chart files. Files that can be handled by AvNav will be uploaded directly into the charts directory and will be available immediately (gemf, mbtiles, xml files).

For charts that will be usable after Conversion you will be asked if they should be put into the input directory of the importer (not on android).

You can optionally select a "set" name - this will become the name of the created gemf file. This way you can add multiple chart files into one gemf file. The state of the conversion can be checked at the status page at "importer".
After an upload to the importer it will still wait some time to allow you to upload more files to the same set. They all will be handled with one conversion.

Tracks

The info dialog for tracks contains some additional information.


Additionally you can start with the  Convert button the conversion of the track to a route. Using the overlay button you can add the track to overlays or remove it from them.
