Some minor modifications to make access to BSH servers working again.

Sources

For the Mobile Atlas Creator I created a coupl of map sources that will give you more control when accessing online map source with some xml config. You need to unpack the file avnav-mapsources.zip within the directory "mapsources" of the Mobile Atlas Creator.

You will get (among others) a "mashUp" with BSH chart services (see bsh-viewer) and OpenSeaMap ("BSH OpenSeaMap 2014 Extended"). Additionally BSH allone ("BSH 2014 Extended") or OpenSeaMap + OpenStreetMap ("OWS OpenSeaMap 2014").

If you want to change - just adapt the .exml files. For the BSH layers you can try them with my bsh-viewer (just edit the sources on the right side). Additionally you can adapt the colors (I tried to get some more contrast). To change the color mappings you can open one of the tiles with a painting program and pick up the hex values for the colors you would like to change. Those must be adapted inside the .exml files.

Typically the download will run for a long time - the BSH servers are slow. If the download stoips, just try again in MOBAC. Set the cache to a sufficient time (e.g. one month). Always use "OsmdroidGEMF" as atlas format.

If you apply changes to the .exml files you need to delete the databases for this chartsource (below tilestore) and restart MOBAC.
