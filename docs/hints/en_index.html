Configuration and Adaptation

There are various ways to adapt AvNav to your personal needs:

  • Adapt various aspects of the WebApp - Settings Page.
  • Adapting the value displays (widgets) on the navigation page and on the dashboard pages: adapting the layout.
  • Adapting the look and feel via css. As the app itself is browser based, you can adapt the look and feel via css stylesheets. This is described at adapting via css
  • Adapting icons (Boat, Ais,...)
  • Adapting the keyboard mappings
  • adapting the server config file (avnav_server.xml) - not on android
  • control the servers wifi connections - Wifi Page - not on Android and OpenPlotter
