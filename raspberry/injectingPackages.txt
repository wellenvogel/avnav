Injecting packages into image:
#prepare binfmt (use ochartsng/docker/prepareAsRoot.sh)
#as root
img=.....img
(1) raspi.sh mount $img
(2) copy packages to raspi-mount/p2/root
(3) chroot raspi-mount/p2
(4) commands:
apt update
apt install ./root/...
(5) if size is to small:
(5.1) raspi2.sh umount $img
(5.1) dd bs=1M count=400 < /dev/zero >> $img
(5.2) losetup -P --show -f $img
(5.3) use gparted to enlarge partion
(5.4) losetup -d xxx #from 5.2
...mount
