## AvNav Entwickler-Versionen (Beta)

Um zwischen den Release-Versionen Updates schon für einen breiten Test bereitzustellen, gibt es Beta Versionen. Diese sind ggf. nicht vollständig getestet - man muss also eventuell mit Fehlern rechnen.

[Hier](../../downloads/daily) geht es zu den Beta Versionen.

Wie man diese Beta Versionen installiert, hängt vom System ab.

### Linux/Raspberry

Man kann die Pakete direkt herunterladen und dann wie unter [Paketinstallation Raspberry](raspberry.md#packages) oder [Paketinstallation Linux](linux.md#packages) beschrieben installieren.

Falls man ein [AvNav Image](raspberry.md#images) nutzt kann man dort auch das Repository für die Beta Versionen einrichten. TODO

## Android

Für Android kann man direkt die APK Datei herunterladen und installieren. Die Applikation heisst dann "AvNav-beta" und kann neben einer normalen AvNav Installation genutzt werdeen.

## Windows

Unter Windows kann die Update Funktion des [Installers](windows.md) genutzt werden und die URL einer der Zip Dateien aus der Liste der Betas bei "Update" eingesetzt werden.
