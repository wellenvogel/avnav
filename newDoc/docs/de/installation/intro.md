# Allgemeines

AvNav steht für verschiedene Plattformen und Betriebssysteme bereit.

| Plattform/Betriebssystem | Anmerkungen |
| --- | --- |
| [Raspberry Pi/Linux](raspberry.md) | Es gibt fertige Images für einen Betrieb mit oder ohne Bildschirm oder man kann Pakete auf einem System installieren. Auch eine Installtion unter OpenPlotter ist möglich. |
| [Linux](linux.md) | Es gibt fertigae debian aund RPM Pakete für die Installation auf verschiedenen Linux Systemen |
| [Android](android.md) | Für Android ist AvNav als App im Store oder zum Download verfügbar |
| [Windows](windows.md) | AvNav kann auch unter Windows laufen. Es gibt dafür einen Installer und die Downloads für die Software |

## Versionen

AvNav wird als fertiges Paket, als installierbare Zip Datei oder als eine APK für Android ausgeliefert. In Abständen gibt es neue "freigegebene" Versionen (Releases). Zwischen den Releases werden Entwickler-Versionen (Beta) bereitgestellt. Diese dienen dem zeitigen Test neuer Funktionen - können aber noch Fehler enthalten.
