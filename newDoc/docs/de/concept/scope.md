# Leistungsumfang: was kann AvNav

Wer eine Navigationssoftware einsetzt, möchte in erster Linie sicher von A nach B gelangen. AvNav bietet dafür mehr als nur die einfache Kartenanzeige und vereint viele Funktionen, für die man klassischerweise mehrere separate Geräte an Bord bräuchte.

Die Grundlage jeder Navigation ist das Kartenmaterial. AvNav ist äußerst flexibel und unterstützt von Haus aus verschiedenste Rasterkarten-Formate. Besonders interessant ist für viele Skipper die nahtlose Integration von aktuellen Vektorkarten. Über das System von [o-charts](https://o-charts.org) lassen sich kostengünstige, hochqualitative Seekarten direkt in AvNav nutzen. Auf der Karte selbst lassen sich Routen intuitiv per Touch planen, Wegepunkte setzen und Distanzen messen. Einmal geplante Törns können gespeichert, exportiert oder direkt an einen angeschlossenen Autopiloten übergeben werden, der das Schiff dann entlang der Route steuert.

Neben der reinen Karte fungiert AvNav als vollwertige Instrumentenanzeige. Wenn die entsprechenden Sensoren des Bootes angebunden sind, lassen sich Daten wie Windrichtung, Windgeschwindigkeit, Wassertiefe oder Logge in übersichtlichen Anzeigefeldern ("widgets") direkt neben der Seekarte oder in eigenen Anzeigepanels ("dashboards") einblenden. Diese Anzeigen kannst Du frei nach Deinen eigenen Wünschen zusammenstellen, sodass Du im Cockpit genau die Werte im Blick hast, die für die aktuelle Situation relevant sind.

Ein wichtiges Sicherheitsfeature ist die Integration von AIS-Daten (Automatic Identification System). Empfängt das Boot Signale anderer Schiffe, stellt AvNav diese zielgenau auf der Karte dar. Die Software berechnet im Hintergrund kontinuierlich, ob man sich einem anderen Schiff gefährlich nähert, und gibt bei kritischen Werten für den geringsten Passierabstand (CPA) und die Zeit bis dorthin (TCPA) optische und akustische Warnungen aus. 

Damit all diese Sensordaten zusammenfließen können, bringt AvNav eine weitere unsichtbare, aber mächtige Funktion mit: es arbeitet als sogenannter NMEA-Multiplexer. Das bedeutet, AvNav kann Daten aus verschiedenen Quellen (wie GPS-Maus, Anemometer, AIS-Empfänger, Tiefenmesser, Logge, ...) sammeln, verarbeiten und an andere Geräte oder Apps weiterleiten. Auch die Anbindung an moderne Schiffsnetzwerke wie SignalK ist problemlos möglich, wodurch AvNav zur zentralen Datendrehscheibe an Bord wird.
