# Was ist AvNav
AvNav - ins Leben gerufen und entwickelt von Andreas Vogel (Wellenvogel) - ist eine ausgereifte Navigationssoftware für den Einsatz auf Sportbooten. Sie bietet all das, was Skipper für eine sichere Törnplanung, die Fahrt und die Orientierung auf dem Wasser benötigen: von der Darstellung nautischer Seekarten über die Einbindung und Anzeige von Bootsinstrumenten, Ankeralarmfunktionen bis hin zur Kollisionswarnung mittels AIS. 

AvNav wurde von Beginn an gezielt für die rauen Bedingungen an Bord entworfen. Im Mittelpunkt steht immer die Maxime, dass eine Navigation im Cockpit – auch bei Wind, Nässe und Schiffsbewegungen – verlässlich und intuitiv über Touch-Displays bedienbar bleiben muss. AvNav schließt damit die Lücke zwischen teuren, fest installierten Plottern der großen Marine-Ausstatter und oft zu simplen Handy-Apps, indem es eine professionelle und vor allem hochgradig anpassbare Navigation für jedermann zugänglich macht.

Ein entscheidendes Merkmal, das AvNav von kommerziellen Produkten unterscheidet, ist seine Natur als Open-Source-Projekt (freie Software). Das bedeutet in erster Linie, dass die Nutzung des Programms - abgesehen von manchen Seekarten - völlig kostenlos ist. Es gibt keine versteckten Abonnements oder gesperrten Premium-Funktionen. Doch Open Source bedeutet noch viel mehr: der gesamte Programmcode ist öffentlich einsehbar und wird transparent auf der Entwicklerplattform [GitHub](https://github.com/wellenvogel/avnav/) verwaltet. Durch diesen offenen Ansatz gibt es keine Herstellerbindung (Vendor-Lock-in). Jeder Programmierer weltweit kann den Code überprüfen, Fehler melden oder beheben, Erweiterungen (Plugins) schreiben oder neue Funktionen beisteuern.

Getragen wird dieses offene Konzept von einer engagierten und hilfsbereiten Community. Das Herzstück des Austauschs, der Fehlerbehebung und der Weiterentwicklung bildet dabei das deutsche [Segeln-Forum](https://www.segeln-forum.de/board/195-open-boat-projects-org/). Hier treffen sich Einsteiger, erfahrene Anwender und die Entwickler selbst auf Augenhöhe. Wenn Du speziellere Fragen zur Installation hast oder Hilfe bei einem komplexen Setup benötigst, ist dieses Forum die beste Anlaufstelle. Viele der heutigen Funktionen von AvNav basieren direkt auf den praktischen Erfahrungen, den Wünschen und dem Feedback der Segler aus dieser Community. AvNav ist somit eine Software aus der Praxis für die Praxis, die durch ihre Nutzer stetig mitwächst.
