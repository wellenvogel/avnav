# What is AvNav
AvNav – initiated and developed by Andreas Vogel (Wellenvogel) – is a mature navigation software designed for use on recreational boats. It offers everything skippers need for safe trip planning, navigation, and orientation on the water: from displaying nautical charts to integrating and displaying boat instruments, anchor alarm functions, and collision warnings using AIS. 

AvNav was designed from the ground up specifically for the harsh conditions on board. At its core is always the principle that cockpit navigation must remain reliable and intuitive to operate via touch displays – even in wind, wet conditions, and ship movement. AvNav thus bridges the gap between expensive, permanently installed chartplotters from major marine electronics brands and often overly simplistic mobile apps, by making professional and, above all, highly customizable navigation accessible to everyone.

A defining feature that sets AvNav apart from commercial products is its nature as an open-source project (free software). First and foremost, this means that using the program – with the exception of certain charts – is completely free. There are no hidden subscriptions or locked premium features. But open source means much more: the entire program code is publicly accessible and transparently managed on the developer platform [GitHub](https://github.com/wellenvogel/avnav/). Thanks to this open approach, there is no vendor lock-in. Any programmer worldwide can review the code, report or fix bugs, write extensions (plugins), or contribute new features.

This open concept is supported by a dedicated and helpful community. The heart of discussions, troubleshooting, and further development is the German [Segeln-Forum](https://www.segeln-forum.de/board/195-open-boat-projects-org/). Here, beginners, experienced users, and the developers themselves meet on equal terms. If you have specific installation questions or need help with a complex setup, this forum is the best place to go. Many of AvNav's current features are based directly on the practical experiences, wishes, and feedback of the sailors in this community. AvNav is thus software created by practical users for practical use, constantly growing alongside its user base.
